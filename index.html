ADVENTURES IN 3D

This is my story of trying to learn 3D as cheaply as Possible

The story so far.

The website started out as an experiment to see if I could create a site purely from html and CSS, which I had been teaching myself and I needed a subject for the website. As we were in Lockdown when I started this project in April 2020, the obvious subject was the 3D projects I had been doing, from tutorials. I started playing around with blender and other free 3D software ages ago. However, either my computer was not powerful enough or I did not understand the programs, so I never got far. But with the new UI in Blender and my updated computer things started to make sense and I got some projects that even looked half decent. This is the story of those projects.
